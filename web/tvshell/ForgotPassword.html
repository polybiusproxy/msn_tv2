If you’ve forgotten the Passport password for you can create a new one after you confirm your identity by providing some information. To create a new password, you’ll need the following information:

  • The state and ZIP code you used when creating your Passport
  • The answer to your secret question
Once we confirm this information, you can create a new password.

To start creating a new password, choose Continue. To return to the Sign-in page without creating a new password, choose Cancel.
