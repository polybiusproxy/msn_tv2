You have reached the Enter wireless network key page because your wireless router or access point requires a special WEP key in order to connect.

Routers typically store four separate keys of the same type and length. Your uses the default key, "key 1."

Here's the easiest and most reliable way to enter a WEP key:

Enter the key (key 1) exactly as it appears on the setup page of your router or access point. (You might find it useful to print your router’s setup page using a computer on your home network.)

Note: Some wireless access points use a passphrase to automatically generate WEP keys in the proper format and length. However, a passphrase is not the same as a WEP key, and will not function in its place.
