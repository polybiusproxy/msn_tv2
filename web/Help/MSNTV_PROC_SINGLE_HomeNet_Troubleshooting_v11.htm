If you're having trouble setting up or using your home network, make sure that:
  • the computer you are trying to access and the network hub/router are both powered on
  • the Ethernet cable (or USB wireless dongle) is properly connected to the
  • your computer is properly connected to your hub/router and is configured correctly
  • file sharing is enabled on your computer
  • the folder you want to access is designated as a shared resource. (For Windows XP Professional users, we recommend that in the settings you set Share Permissions to Everyone)
  • The username and password you are entering match those assigned on the computer (and/or the folder you're trying to access). These are not the user name and password for your MSN TV account.
  • For more information about your home network setup, go to www.msntv.com
