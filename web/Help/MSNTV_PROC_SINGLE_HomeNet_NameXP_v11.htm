To find your computer's name in Windows XP, follow these steps:
  1. Right-click My Computer, and then choose Properties.
  2. Click the Computer Name tab in the System Properties window that appears.
Important: You need to use the information in the Full computer name, not the Computer description. If the full computer name contains domain information (for example, MyLaptop.mydomain.mycompany.com), disregard everything after the first period. In this example, the computer name is MyLaptop.
