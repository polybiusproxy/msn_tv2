The blue Messages light on the front of your Internet & Media Player can tell you when you have new e-mail. It can even check for new mail when you're signed out. But you'll have to set it up.

Note: If you want your Messages light to work while you're signed out, you must save your password on the Sign-in page.

  1. Choose Settings from the Sign-in page.
  2. Choose Set up automatic updates.
  3. Check the box labeled Check for new mail while I'm signed out.
  4. Choose the box to the right of Check for new mail every and press OK to see a list of time intervals.
  5. Choose how often you want the Player to check for new mail. (If you are using a dial-up connection, this involves a phone call to the MSN TV Service each time.) You can also choose when the Player will start checking for mail. Move the yellow highlight box to the line that says Check for updates around and choose a time.
  6. Choose Save Changes.
  7. On the Sign-in page, enter your password and make sure to check the box that says Save password so I don't need to retype it.

The blue light will illuminate when a person who is signed in gets new mail. If no one is signed in, then it lights up when anyone with a saved password gets mail. In that case, the Sign-in page will display the words New mail next to whoever has mail waiting.
