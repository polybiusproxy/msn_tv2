In order to use your to access files that are on your computer, you must first enable file sharing on your computer.

First, determine which operating system (OS) your computer is using. (If you're not sure, go to your computer's online Help and search for OS.)

Choose your computer's OS below to learn how to set up file sharing:

Windows 98
Windows Millennium Edition (Me)
Windows XP
  1. Click Start, point to Settings, and then click Control Panel.
  2. Double-click the Network icon, or use the previously created shortcut on the desktop.
  3. Click the Network Identification tab.
